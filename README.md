## 🔎 Overview
Designed and implemented a volunteer management database and analytics platform for the Olympic Games.<br/>

The project demonstrates the complete data lifecycle, from relational database design to business intelligence reporting, enabling data-driven decision-making for large-scale event operations.

Project Objectives:<br/>
▪️ Manage volunteer information and assignments<br/>
▪️ Track training, shifts, events, and roles<br/>
▪️ Support operational planning and resource allocation<br/>
▪️ Transform transactional data into analytical insights<br/>
▪️ Visualise performance through Power BI dashboards<br/>
